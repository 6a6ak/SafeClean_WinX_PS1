SafeClean - Laptop Deployment Package 
====================================== 
 
QUICK START: 
1. Run: LAPTOP-QUICK-START.bat 
2. If Java errors, run: INSTALL-JAVA-ON-LAPTOP.bat 
3. For detailed help, see: LAPTOP-DEPLOYMENT-GUIDE.md 
 
MAIN APPLICATION: 
- SafeClean-Safe.bat (recommended launcher) 
- SafeClean-2.0.0.jar (core application) 
- SafeClean-Setup.exe (installer - may trigger antivirus) 
 
TROUBLESHOOTING: 
- Simple-Java-Test.bat (basic Java check) 
- Java-Diagnostics.bat (comprehensive testing) 
 
REQUIREMENTS: 
- Windows 7/8/10/11 
- Java 8+ (Java 17+ recommended) 
- Administrator privileges (recommended)
